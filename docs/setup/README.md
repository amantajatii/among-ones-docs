# Getting Started

The AmongOnes repository is divided into several main components, each operating independently but integrated via APIs and WebSockets. To set up the ecosystem locally, you will need to start the backend, the frontend, and compile/deploy the smart contracts.

## Components Breakdown

| Directory | Component | Tech Stack |
|-----------|-----------|------------|
| `among-ones-sc` | Smart Contracts | Solidity, Foundry, One Chain Network |
| `among-ones-be` | Backend Game Server | Node.js, Express, Socket.io, TypeScript |
| `among-ones-fe` | Frontend Client | Next.js 16, React 19, Tailwind CSS 4, Wagmi |
| `amongOnesMove` | Move Contracts | Move, Sui/Aptos (Skeleton/Draft) |

Please navigate to the sub-pages for specific instructions on how to install and run each individual component.
